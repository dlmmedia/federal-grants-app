DARK MODE
---------

Implementation:

- System preference detection enabled

Theme Toggle:
- Sun icon for light mode
- Moon icon for dark mode
- Dropdown with Light/Dark/System options

ACCESSIBILITY
-------------

Focus States:
- Ring: 3px ring with Ring color
- Outline: None (using ring instead)

Color Contrast:
- All text meets WCAG AA standards
- Interactive elements have clear focus states

Keyboard Navigation:
- All interactive elements keyboard accessible
- Logical tab order
- Skip links where appropriate

Screen Readers:
- Semantic HTML
- ARIA labels on icon-only buttons
- Descriptive link text

RESPONSIVE DESIGN
-----------------

Breakpoints:
- SM: 640px
- MD: 768px
- LG: 1024px
- XL: 1280px
- 2XL: 1536px

Mobile-First Approach:
- Base styles for mobile
- Progressive enhancement for larger screens
- Touch-friendly targets (min 44x44px)

Grid Layouts:
- 1 column on mobile
- 2 columns on tablet (md:)
- 3 columns on desktop (lg:)
- 4 columns on large desktop (xl:)

ANIMATION & TRANSITIONS
-----------------------

Transition Duration:
- Fast: 150ms
- Default: 300ms (cubic-bezier(0.4, 0, 0.2, 1))
- Slow: 500ms

Transition Timing:
- Default: cubic-bezier(0.4, 0, 0.2, 1)
- Enter: ease-out
- Exit: ease-in

Custom Animations:
- fadeIn: Fade in with upward movement (0.6s)
- slideIn: Slide in from left (0.6s)
- scaleIn: Scale up from 95% (0.5s)

Common Animations:
- Hover: scale, shadow, opacity, translateY(-4px)
- Focus: ring appearance
- Loading: spin, pulse
- Enter/Exit: fade, slide
- Testimonials: Carousel with slide transitions (5s auto-rotate)
- Logo hover: Grayscale to color transition
- Button hover: Scale and shadow enhancement

Smooth Scrolling:
- Enabled globally with scroll-behavior: smooth
- Anchor links animate smoothly

VOICE & TONE
------------

Brand Voice:
- Professional yet approachable
- Clear and concise
- Helpful and supportive
- Trustworthy and reliable

Content Guidelines:
- Use active voice
- Be specific and concrete
- Avoid jargon when possible
- Provide context and examples
- Focus on user benefits

Error Messages:
- Clear explanation of what went wrong
- Actionable next steps
- Supportive tone

Success Messages:
- Confirm action completed
- Brief and positive
- Use checkmark icon

LOGO & BRANDING
---------------

Logo:
- Icon: "GS" in rounded square with primary background
- Text: "GrantScope" (proper case, bold)
- Color: Primary (Coral/Salmon)
- Font: Bold, tracking-tight
- Icon + text combination

Logo Usage:
- Minimum size: 120px width
- Clear space: Equal to height of "G"
- Always link to home page
- Icon is 32px square (w-8 h-8)

Tagline:
"The complete grant platform"

IMAGERY
-------

Image Style:
- Professional and clean
- High quality
- Relevant to grant/government context
- Diverse and inclusive
- Large feature images with rounded corners (rounded-2xl)
- Shadow effects for depth (shadow-2xl)

Image Organization:
- /images/backgrounds/ - Background images and patterns
- /images/customer-logos/ - Customer organization logos
- /images/features/ - Feature screenshots and visuals
- /images/testimonials/ - Customer testimonial photos
- /images/funders/ - Major funder logos
- /images/social-media/ - Social media icons

Image Effects:
- Grayscale filter on logos (hover to reveal color)
- Opacity transitions (60% to 100% on hover)
- Border and shadow on important images
- Aspect ratio maintained with Next.js Image component

Placeholder Colors:
- Use muted backgrounds
- Skeleton loaders for loading states

Icons:
- Lucide React icon library
- Consistent sizing
- Simple, not overly colorful
- Meaningful and recognizable

Decorative Elements:
- SVG underlines on key headlines (animated curves)
- Gradient blob backgrounds (blur-3xl)
- Play button overlays for video previews
- Star ratings for testimonials

1. Hero Section:
